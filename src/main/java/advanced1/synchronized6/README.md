6. 동기화 - synchronized

6-1 임계 영역
```html
임계 영역은 영어로 크리티컬 섹션이라 한다.
여러 스레드가 동시 접근하면 데이터 불일치나 예상치 못한 동작이 발생할 수 있는 위험하고 또 중요한
코드 부분을 뜻한다.
여러 스레드가 동시에 접근해서는 안 되는 공유 자원을 접근하거나 수정하는 부분을 의미한다.
임계 영역은 한 번에 하나의 스레드만 접근할 수 있도록 안전하게 보호해야 한다.
자바는 synchronized 키워드를 통해 아주 간단하게 임계 영역을 보호할 수 있다.
하나의 스레드만 실행할 수 있는 안전한 임계영역은 가능한 최소한의 범위에 적용해야 한다. 그래야 전체적인 처리 성능을 더 높일 수 있다.
이런 동기화를 사용하면 레이스 컨디션과 데이터 일관성의 문제들을 해결할 수 있다.
```
