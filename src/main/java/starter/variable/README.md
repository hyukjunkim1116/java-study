1.변수

1-1 변수 사용 이유 
```
20대신 30을 출력하고자 할때, 여러 파일에서 수정해야 한다면? 사용자가 숫자를 입력하고 출력하고 싶다면?
여러 파일에 수정하는 작업은 번거롭다. 그리고 사용자가 입력하는 숫자는 계속 바뀐다.
따라서 어딘가에 값을 보관하고 꺼내서 읽을 수 있어야 한다. 이를 위해 변수(variable)를 활용한다.
```
1-2 변수란?
```html
int a : 숫자 정수를 보관할 수 있는 이름이 a라는 데이터 저장소 이걸 변수라 한다.
a = 10 : 자바에서 = 는 오른쪽에 있는 값을 왼쪽에 저장한다는 뜻이다. 변수에 처음으로 값을 대입해서 저장하는 것을 변수 초기화 라고 한다.
System.out.println(a) : 변수 값 읽기. 변수 값은 반복해서 읽을 수 있다.
```

1-4 변수 선언과 초기화
```html
변수를 선언하면 컴퓨터의 메모리 공간을 확보해서 그곳에 데이터 저장 가능. 그리고 변수의 이름을 통해서 해당 메모리에 접근 가능
선언한 변수에 처음으로 값을 저장하는 것을 변수 초기화라고 부른다.
변수는 초기화 해야한다. 하지 않으면 컴파일 에러가 발생한다. 왜? 컴퓨터에서 메모리는 여러 시스템이 함께 사용하는 공간인데, 변수를 선언하면 메모리상의 공간의 일부를 차지한다. 그 공간이 원래 어떤 값이 있었는지 모르기 때문에 초기화 하지 않으면 이상한 값이 나올 수 있다.
```

```html
int: 정수
double: 실수
boolean: 참,거짓
char: 문자 한개
String: 문자열
int a =100; 에서 100같은 고정 된 값을 리터럴(literal)라고 부른다.
```
1-6 변수 타입 2
```html
정수형
byte: ~128-127(2의 8제곱)
short: -32768-32767(2의 16제곱)
int: -2147483648-2147483647(2의 32제곱, 약 20억)
long: -9223372036854775808-9223372036854775807(2의 64제곱,8byte)
실수형
float: 2의 32제곱
double: 2의 64제곱
기타
boolean: 1byte
char: 문자하나(2byte)
String: 동적으로 지정
리터럴 타입 지정: int가 기본, int범위 넘어서면 L붙여서 정수 리터럴을 long로 변경해야한다. 실수 리터럴은 double이 기본. float  쓰려면 f 붙이자.
```
