1. 클래스와 데이터

1-1 클래스를 사용하는 이유
```html
사람이 코드를 관리하기 좋은 방식은 개념을 하나로 묶는 것이다. 따라서 클래스를 사용한다.
클래스에 정의한 변수들을 멤버 변수 또는 필드라고 한다.
타입은 데이터의 종류나 형태를 나타낸다. int,String 등이 있다. 클래스를 사용하면 타입을 직접 만들 수 있다.
사용자 정의 타입을 만드려면 설계도가 필요하고 이를 클래스라고 말한다.
클래스를 사용해서 실제 메모리에 만들어진 실체를 객체 또는 인스턴스.
new Student()를 사용해서 메모리에 Student 객체를 생성한다. 
객체를 생성하면 자바는 메모리 어딘가에 있는 객체에 접근할 수 있는 참조값을 반환한다.
```
1-2 객체 사용
```html
클래스를 통해 만든 객체를 접근하려면 .(점)을 사용하면 된다.
```
1-3 용어 정리
```html
클래스 : 객체가 가지는 변수와 메서드를 정의한다. 붕어빵 틀.
객체 : 클래스에서 정의한 속성관 기능을 가진 실체. 객체끼리 독립적인 상태를 가짐. 참조값이 다르기 떄문
인스턴스 : 특정 클래스로부터 생성된 객체. 객체와 혼용된다.
둘다 클래스에서 나온 실체 but 인스턴스는 객체보다 좀 더 관계에 초점을 맞춤.
```
